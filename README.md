# enterprise-data-platform (mono-repo)

This repository now contains two separate projects under one repository root:

- `enterprise-data-platform-app/`
  - The original enterprise data platform project files
  - Includes Airflow, Spark, UI, and data pipeline configuration

- `parent-communication-bot/`
  - AI Parent Communication Bot project
  - Sends personalized WhatsApp summaries and answers parent queries

## How to use

Open the project folder you want to work on:

- `enterprise-data-platform-app/` for the platform project
- `parent-communication-bot/` for the WhatsApp parent bot

Each project has its own README and setup instructions.
